# _{Adress Book}_

##### _{Saves Contacts into an Adress Book}, {08/07/2015}_

#### By _**{Julio Diaz}**_

## Description

_{Creates an Address Book that alows users to fill in information about contacts}_

## Setup

* Open Terminal
* Enter the web directory in AddressBook using terminal
* Enter the following command into termianl 'php -S localhost:8000'
* Open a web browser and type this into the address bar 'localhost:8000'

## Technologies Used

_{Php, Silex, Twig, Composer}_
